## Data Cleaning and Preprocessing - Task 1

### Objective:

To clean and prepare a raw dataset by handling missing values, duplicates, and inconsistent formats.

### Dataset:

Netflix Movies and TV Shows Dataset

### Steps Performed:

* Identified missing values using `.isnull()`
* Filled missing values in director, cast, and country columns
* Removed rows with missing date_added
* Removed duplicate records
* Standardized text values
* Converted date column to datetime format
* Renamed column headers

### Tools Used:

Python (Pandas), Google Colab

### Result:

The dataset is cleaned and ready for analysis.
